
# ContactRise #
- Worked with MongoDB. There is no migration, only data.
- The communication of Microservices was done over HTTPClient.
- The connection settings from the app setting file according to you.

# Technologies #
- .net Core 5.0
- MongoDB
- RabbitMQ
- AutoMapper
- MediatR
- CQRS
- Microservices
